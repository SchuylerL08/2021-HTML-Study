Redstone Comparator


Block ID 404
Stack Size 64
Blast Resistance 0

Uses:
  • Brewing: No use
  • Crafting: No use
  • Recipe: Three stone blocks, three redstone torches, and one nether quartz
  • Circuits:
    • Can generate a redstone signal based on how full a container is that the comparator is placed against
    • Can be used to maintain a signal strength allowing the redstone signal to pass through it without
      losing or gaining any strength
    • In compare mode a comparator compares its rear input to its two side inputs. If either side input is
      greater than the rear input, the comparator output turns off. If neither side input is greater than the
      rear input, the comparator outputs the same signal strength as its rear input.
    • In subtract mode a comparator subtracts the signal strength of the higher side input from the signal
      strength of the rear input.
